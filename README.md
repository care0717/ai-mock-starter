# AI mock starter

AIと一緒にWebモックを作るためのテンプレートです。ビルドすると**1つのHTMLファイル**になるので、そのまま共有できます。

## 使い方

```bash
# 1. クローンする
git clone <このリポジトリのURL>
cd mock-template

# 2. `npm install`で依存をインストール
npm install

# 3. AIに頼んで好きにモックを編集・作成

# 4. 完成したら`npm run build`でビルド（dist/index.html が生成されます。それをブラウザで開いて確認したり、そのまま納品できます。）
npm run build

# おまけ便利機能
# 開発サーバーを起動（ブラウザで自動的に開きます。ファイルを編集したら即座に反映されます。）
npm run dev
```

## ファイルの編集方法
基本AIに頼みましょう。自分でやる場合は以下を参照。

| やりたいこと | 編集するファイル |
|---|---|
| ページを追加したい | `src/pages/` に新ファイル → `src/main.js` でルート追加 |
| ヘッダーを変えたい | `src/components/Header.html` |
| フッターを変えたい | `src/components/Footer.html` |
| 共通パーツを追加したい | `src/components/` に新ファイルを作成 |

## AIに頼むときのコツ

- 「ヘッダーのナビゲーションにログインボタンを追加して」のように、**どのパーツ**を**どう変えたいか**を伝えると正確です
- 「新しいお問い合わせフォームのコンポーネントを作って」のように、新パーツの追加もお願いできます
- 最後に `npm run build` でビルドして、`dist/index.html` を共有しましょう
